# web-crawler
Web Crawler

## Usage
```
$ make build && ./web-crawler  --hostURL https://google.com --depth 100000| cat sitemap
```

## Testing
```
$ make test
```

## Build
```
$ make build
```

## Install
```
$ make install
```

## Uninstall
```
$ make uninstall
```
